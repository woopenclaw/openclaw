# Prompt 模板 — 小说章节生成

## 首段（第一章）模板

```
你是网络小说主笔，正在写第{N}章：{标题}。

【本章剧情】{大纲剧情}
{角色信息}

硬性要求：
1. 纯中文！所有数字用中文写法（一、二、三十），绝不出现阿拉伯数字和英文
2. 零AI标记词汇！不得出现"暗思涌起""心理描写""环境描写""动作描写""命运的齿轮"等
3. 禁止模板化结尾！不要"真正的危机才刚刚开始""未完待续"
4. 主角名{主角名}，全文统一
5. {金手指描述}
6. 直接输出小说正文，不要任何说明文字

字数必须达到12000字。
```

## 续段（第2+段）模板

```
你是网络小说主笔，继续写第{N}章：{标题}。

【本章剧情】{大纲剧情}

【已有内容结尾】
{结尾300-500字摘录}...

【核心约束】人物、世界观、金手指、剧情走向必须与已有内容连贯。

从已有内容的结尾处**自然续写**，不可重复已有内容。

硬性要求：
1. 纯中文！数字用中文写法，零英文零阿拉伯数字
2. 零AI标记词汇
3. 禁止模板化结尾
4. 直接输出续写正文

续写内容必须至少{需要字数}字。
```

## 硬编码约束（注入所有 prompt）

```
1. 纯中文！所有数字用中文写法(一、二、三十)，绝不出现阿拉伯数字和英文
2. 零AI标记词汇（不得出现"暗思涌起""心理描写""环境描写""动作描写""命运齿轮"等）
3. 禁止模板化结尾（"真正的危机才刚刚开始""未完待续"等）
4. 主角姓名统一，不可切换
5. 金手指描述一致
6. 忠于大纲，不自由发挥核心剧情
7. 直接输出正文，不要说明
```

## 大纲格式要求

大纲 JSON：
```json
{
  "chapters": {
    "1": {"title": "废物少年", "plot": "主角出场，被退婚..."},
    "2": {"title": "退婚之辱", "plot": "退婚现场，主角立誓..."},
  }
}
```

## Prompt 变量注入

| {{变量}} | 说明 | 示例 |
|---------|------|------|
| `{{主角名}}` | 主角名字 | 林阎 |
| `{{金手指}}` | 核心设定 | 一枚神秘铜钱 |
| `{{卷号}}` | 当前卷 | 第一卷 |
| `{{大纲剧情}}` | 本章大纲 | 来自大纲文件 |
| `{{角色信息}}` | 本章出场角色 | 林阎、柳如烟 |
| `{{需要字数}}` | 计算出的目标 | 4000-6000 |
