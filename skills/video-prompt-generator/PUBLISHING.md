## Required credential

This skill requires exactly one credential:

- `NANOPHOTO_API_KEY` — required

Do not paste the API key into chat. Configure it through the platform's secure environment-variable setting for this skill.

## Safety / usage notes

- Only submit topics and image URLs you are authorized to use.
- For image-to-video mode, only provide public image URLs you are comfortable sending to the NanoPhoto service.
- Do not provide private or sensitive images as public URLs.
- This skill does not ask for unrelated credentials.
- This skill does not read messaging credentials or send external notifications.
