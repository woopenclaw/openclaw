# 2026-06-18 Session Memory

## 鼠标键盘控制能力建立

### 核心发现
- WSL → Windows PowerShell 直连通路已验证通过
- 坐标换算公式：`实际坐标 = 截图显示坐标 × 1.28`（截图2560×1440，显示2000×1125）
- 输入框精准定位：OpenClaw Control 输入框在 (639, 1303)

### 工具脚本
- `/mnt/c/Temp/win_input.ps1` — 单次操作（鼠标/键盘/截图）
- `/mnt/c/Temp/win_batch.ps1` — 批量操作（JSON命令序列，0.2s/操作）
- `~/scripts/win_control.sh` — WSL侧封装脚本
- `~/scripts/quick_control.sh` — 快速调用接口
