# Checkpoints — 关键决策存档

> 每次做出重要配置变更、架构决策、项目方向调整时，创建一个 checkpoint。
> 格式：YYYY-MM-DD-<简短描述>.md

## Checkpoint 模板

```markdown
# Checkpoint: [简短描述]

- **时间**: YYYY-MM-DD HH:MM
- **决策人**: 飞哥 / 自主
- **背景**: [为什么做这个决策]
- **决策内容**: [做了什么]
- **影响范围**: [哪些系统受影响]
- **回滚方案**: [如果出问题怎么恢复]
- **验证方式**: [怎么确认改动生效]
```
